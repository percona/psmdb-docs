    Percona Server for MongoDB 4.2.9-10 (2020-10-09)¶

    Release date October 9, 2020
    Installation Installing Percona Server for MongoDB

    Percona Server for MongoDB 4.2.9-10 is an enhanced, source-available, and highly-scalable database that is a fully-compatible, drop-in replacement for MongoDB 4.2.9 Community Edition. It supports MongoDB 4.2.9-9 protocols and drivers.

    This release fixes security vulnerability CVE-2020-26542.

    Improvements¶

    • PSMDB-674: Provide binary tarball with shared libs and glibc suffix
